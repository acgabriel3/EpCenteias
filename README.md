
Pacote de Epidemiologia Do Centeias (PEpCenteias) ---renoemar

O projeto tem como objetivos a criação de uma biblioteca de avaliação da qualidade de dados de saúde.
the project have how objective the creation of a library of data health quality.

Framework

O projeto propõe um novo framework para a arquitetura do programa em 'forContributors', que é automaticamente transformado no formato R package. Tem como objetivo a modularização, manutenibilidade, e flexibilidade facilitando que a comunidade adicione código facilmente ao projeto.
the projet propose a new framework for the architecture of the program in 'forContributors', that is automatcally transformed in the R packege format. Have like objective the modulazization, manutanability and flexibility. Make easy to the community the adiotion of code in the project.

Funções

As funções ainda não possuem o melhor refinamento, sendo que esperamos a ajuda e interesse da comunidade para melhorar as funções do pacote.
The functions don't have yet the better code, then we wait the help and interest of the community to improve the functions of the packege.
